# JavaScript Warm Up

Ce dossier contient une série d'exercices d'introduction à JavaScript. Chaque script aborde un concept fondamental du langage, idéal pour débuter ou réviser les bases.

## Contenu des fichiers

- **0-javascript_is_amazing.js** : Affiche un message simple en JavaScript.
- **1-multi_languages.js** : Affiche plusieurs lignes, chacune dans une langue différente.
- **2-arguments.js** : Affiche le nombre d'arguments passés au script.
- **3-value_argument.js** : Affiche la première valeur passée en argument.
- **4-concat.js** : Concatène deux arguments et les affiche.
- **5-to_integer.js** : Convertit un argument en entier et l'affiche.
- **6-multi_languages_loop.js** : Affiche plusieurs lignes à l'aide d'une boucle.
- **7-multi_c.js** : Affiche un message un certain nombre de fois selon l'argument.
- **8-square.js** : Affiche un carré de taille donnée avec le caractère "X".
- **9-add.js** : Additionne deux nombres passés en arguments.
- **10-factorial.js** : Calcule la factorielle d'un nombre.
- **11-second_biggest.js** : Affiche le deuxième plus grand nombre parmi les arguments.
- **12-object.js** : Crée et affiche un objet simple.
- **13-add.js** : Définit une fonction d'addition exportable.

## Exécution

Chaque script peut être exécuté avec Node.js :

```bash
node <nom_du_script.js> [arguments]
```

Exemple :

```bash
node 0-javascript_is_amazing.js
```
